## Series Circuit

In a series circuit, components are connected one after another.

Key Points:
- Same current flows through all components
- Total resistance = R1 + R2 + R3 + ...

Formula:
R_total = R1 + R2 + R3

Understanding:
- If more resistors are added, total resistance increases
- Current decreases when resistance increases

Example:
If R1 = 2Ω and R2 = 3Ω
Then total resistance = 5Ω
