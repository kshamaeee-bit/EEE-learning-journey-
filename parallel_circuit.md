## Parallel Circuit

In a parallel circuit, components are connected across the same two nodes.

Key Points:
- Voltage is same across all components
- Current divides among branches

Formula:
1 / R_total = 1/R1 + 1/R2 + 1/R3

Understanding:
- Adding more branches decreases total resistance
- More current flows when resistance decreases

Example:
If R1 = 2Ω and R2 = 2Ω
Then:
1/R_total = 1/2 + 1/2 = 1
So, R_total = 1Ω
